# Core Ephemeris Model

This file defines the public research-facing ephemeris concept for Planetary Psychology™.

## Purpose

The Core Ephemeris Model is intended to organize planetary, temporal, atmospheric, and environmental variables for research use.

It is not based on any private birth chart, personal birth time, or individual natal data.

## Public Variables

Potential variables include:

- Date
- UTC timestamp
- Local timezone
- Lunar phase
- Solar activity
- Geomagnetic index
- Seasonal position
- Day length
- Atmospheric pressure
- AQI
- Temperature
- Humidity

## Research Use

The model may be used to explore whether measurable temporal, environmental, and planetary variables correlate with psychological, behavioral, or collective-pattern data.

## Privacy Boundary

No personal birth data, private natal chart data, medical records, or identifying participant information should be committed to this repository.
